# Little Soul

**GitHub-native website builder — themes, layouts, sections, pages. No server. No database. No subscription.**

A single HTML file that lets you build complete multi-page websites and publish them directly to GitHub Pages via the GitHub API — from your browser, with no installation required.

---

## What It Does

Little Soul is a visual website builder that outputs clean, standalone HTML files. Choose a theme, pick a layout, add sections, add pages, publish. Everything happens in the browser. Nothing is sent anywhere except the GitHub API when you publish.

**Six Themes**
Each theme generates its own complete embedded CSS with matched typography, colors, and layout:
- **Forge** — dark terminal, green on black
- **Broadsheet** — editorial newspaper, refined serif
- **Studio** — clean minimal, generous white space
- **Dusk** — warm amber, cozy and inviting
- **Blueprint** — technical monospace, blueprint blue
- **Neon** — dark with vivid accent colors

**Six Layouts**
- Top Nav
- Sidebar Left
- Sidebar Right
- Full Width
- Landing Page
- Docs

**Eight Section Types**
- Hero — title, subtitle, CTA button, optional background image
- Text Block — heading and body copy
- Two Column — configurable ratio, text and/or images
- Image — full width or contained, optional caption and link
- Cards — 2, 3, or 4 column card grid
- Call to Action — centered banner with primary and secondary buttons
- Footer — multi-column link grid with copyright
- Custom HTML — raw HTML injection for anything else

**Multi-Page**
Add unlimited pages. Navigation generates automatically across all pages. Each page has its own filename, title, and meta description.

---

## Usage

1. Open `little-soul.html` in Chrome
2. Choose a theme and layout
3. Add and edit sections
4. Add pages as needed
5. Enter your GitHub username, repository, and personal access token in the right panel
6. Click **PUBLISH**

All pages push to GitHub simultaneously. A CNAME file is generated automatically if you set a custom domain.

---

## No Installation

One HTML file. Open it. Build. Publish. Done.

No Node.js. No npm. No build tools. No server. No subscription. No plugin updates.

---

## Export

Don't want to use GitHub? Click **EXPORT HTML** to download each page as a standalone HTML file. Host it anywhere.
